# Provisional Fixture Policy

Unapproved operational values use:

```text
proposed_default
not_production_approved
```

This applies to professional role assignments, PM technical-review permission, receipt-evidence verifier policy, uploader/verifier exceptions, waiver authority, client evidence criteria, timer durations, business calendars and retention periods.

## Prohibitions

```text
no real staff name as default authority
no client profile labelled approved without record
no production seed using proposed fixtures
no hidden fallback to latest policy
```

Pilot and production gates fail while a required policy remains provisional.
