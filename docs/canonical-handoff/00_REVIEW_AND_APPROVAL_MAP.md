# Review and Approval Map

## Review lanes

### Product Owner / Management
Review product boundary, five-path meaning, management attention and authorization gates.

### Project Manager
Review Work Thread ownership, due changes, outcome acceptance, Submission approval and correction authority.

### Technical Reviewer
Review exact Revision identity, competency, independence and approval/revision/rejection semantics.

### Administration / Document Control
Review manifest preparation, dispatch approval, recipient snapshot, evidence collection and replacement paths.

### Dispatch Coordinator
Review custody attempt lifecycle, external capability scope, failure and re-dispatch.

### Receipt Evidence Verifier
Review evidence item-set freezing, criteria, professional verification, rejection and invalidation.

### QS / Finance
Review Claim Requirements, evidence links, gaps, waiver, ReadyForQSReview and QSVerified.

### System / Security / Operations
Review runtimes, RLS, secret boundaries, outbox, timers, recovery and commissioning.
